# Текстовый квест 

Сюжет: Вы идете по городскому рынку, вас подзывает мужчина и предлагает сыграть в наперстки 
1. У вас есть 3 варианта ответа: 1. согласиться сыграть, 2.пройти мимо, 3. предложить сыграть мужчине в карты 
2. Если вы соглашаетесь играть, мужчина предлагает выбрать 1 из 3 наперстков, под которым предположительно лежит шарик 
3. Суть игры: шарик может находиться только под одним из трех наперстков. 
Если угадал под каким по счету наперстком находится шарик - победил. Если не угадал - увы.
